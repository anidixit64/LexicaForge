LexicaForge NLP Toolkit

A powerful Natural Language Processing toolkit available in multiple programming languages.

Features

  • Text tokenization
  • Entity extraction
  • Text similarity calculation
  • Multi-language support (Python, Java, PHP)
  • Efficient caching mechanisms

Installation

Python

pip install lexicaforge

Java

<dependency>
    <groupId>com.lexicaforge</groupId>
    <artifactId>lexicaforge</artifactId>
    <version>1.0.0</version>
</dependency>

PHP

composer require lexicaforge/lexicaforge

Examples

Python

from lexicaforge import LexicaForge

nlp = LexicaForge()
tokens = nlp.tokenize("Hello World")
entities = nlp.extractEntities("John works at Google")
similarity = nlp.calculateSimilarity("Hello World", "Hello Universe")

Java

LexicaForge nlp = new LexicaForge();
List<String> tokens = nlp.tokenize("Hello World");
List<String> entities = nlp.extractEntities("John works at Google");
double similarity = nlp.calculateSimilarity("Hello World", "Hello Universe");

PHP

$nlp = new LexicaForge();
$tokens = $nlp->tokenize("Hello World");
$entities = $nlp->extractEntities("John works at Google");
$similarity = $nlp->calculateSimilarity("Hello World", "Hello Universe");
